# LVGLSharp.Runtime.Headless

`LVGLSharp.Runtime.Headless` provides the offscreen runtime used for snapshots, automation, and non-interactive rendering flows.

## What it includes

- `OffscreenView`
- PNG export helpers
- snapshot-oriented testing and regression support
- reusable frame sources for remote rendering scenarios

Reference this package directly when you need screenshots, CI-friendly rendering, or headless validation paths.

Docs and package guidance: <https://lvglsharp.net/en/nuget.html>  
Repository: <https://github.com/IoTSharp/LVGLSharp>
