# LVGLSharp.Runtime.Linux

`LVGLSharp.Runtime.Linux` provides the Linux host runtime used by LVGLSharp applications.

## What it includes

- `LinuxView` host selection
- WSLg and X11 desktop paths
- Wayland and SDL host implementations
- FrameBuffer support and DRM/KMS scaffolding

Reference this package together with `LVGLSharp.Forms` when you want to target Linux desktop or device-oriented host paths.

Docs and package guidance: <https://lvglsharp.net/en/nuget.html>  
Repository: <https://github.com/IoTSharp/LVGLSharp>
