# LVGLSharp.Runtime.MacOs

`LVGLSharp.Runtime.MacOs` defines the macOS runtime boundary for LVGLSharp.

## Current scope

- `MacOsView` and runtime options
- diagnostics and host-context types
- surface and frame-buffer scaffolding for future host work

This package is published so the runtime boundary is explicit, but the native macOS host implementation is still evolving. Treat it as an early package for experimentation and integration planning rather than a fully mature runtime.

Docs and package guidance: <https://lvglsharp.net/en/nuget.html>  
Repository: <https://github.com/IoTSharp/LVGLSharp>
