# LVGLSharp.Runtime.Remote

`LVGLSharp.Runtime.Remote` contains the remote-session abstractions used for VNC, RDP, and related rendering paths.

## What it includes

- remote session and runtime factory types
- frame models and frame encoding helpers
- remote input abstractions
- VNC transport pieces and RDP skeleton support
- headless-to-remote frame-source reuse

Use this package when you are building remote display scenarios on top of LVGLSharp.

Docs and package guidance: <https://lvglsharp.net/en/nuget.html>  
Repository: <https://github.com/IoTSharp/LVGLSharp>
