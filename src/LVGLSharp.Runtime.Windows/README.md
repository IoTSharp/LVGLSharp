# LVGLSharp.Runtime.Windows

`LVGLSharp.Runtime.Windows` provides the Windows host runtime used by LVGLSharp applications.

## What it includes

- `Win32View` and related host integration
- Windows-oriented image source support
- generated runtime registration support for application startup

Reference this package together with `LVGLSharp.Forms` when you want the LVGLSharp runtime path on Windows.

Docs and package guidance: <https://lvglsharp.net/en/nuget.html>  
Repository: <https://github.com/IoTSharp/LVGLSharp>
